#### Collection joined with table
```php
<?php
$joinConditions = 'u.field1 = e.field1';
$prodCollection->getSelect()->join(
	['u' => 'table1'],
	$joinConditions,
	[]
);
//Multiple conditions
$joinConditions[] = "u.sku = e.sku";
$joinConditions[] = "e.type_id= 'simple' ";
$joinConditions = implode(
	' AND ', $joinConditions
);
```
